### neovim
## Required
neovim
git
## Installation
# Required
```
mv ~/.config/nvim{,.bak}
```
# Optional but recommended
```
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
```
# Clone
```
git clone https://github.com/Psychopath21312/nvim-lua-backup.git ~/.config/nvim
rm -rf ~/.config/nvim/.git
```
## Usage
`nvim`
